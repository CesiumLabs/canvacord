# Canvacord Assets

This branch holds all the assets required by Canvacord.

# How to add assets?

* Clone this branch: `git clone https://github.com/DevAndromeda/canvacord.git -b assets`
* Add fonts/images to `data` dir
* Commit changes
* Make a pull request
